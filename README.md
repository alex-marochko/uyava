# Uyava (public)

This repository hosts the public Uyava surface:
- `site/` for the website + documentation.
- `packages/uyava_protocol` shared protocol and validation helpers.
- `packages/uyava_core` graph/domain core logic.
- `packages/uyava_ui` shared Flutter UI components.
- `packages/uyava` app-side SDK + embedded DevTools assets.
- `packages/uyava_devtools_extension` DevTools extension.
- `examples/uyava_example` example app for integration/smoke testing.
- `tools/ide_plugins/*` IDE launch plugins.

Private desktop/commercial code stays in the separate `private` repository.
